El objetivo de este proyecto es desplegar una arquitectura web de Alta Disponibilidad y tolerante a fallos en AWS utilizando Infrastructure as Code (IaC) con CloudFormation. La solución levanta un Application Load Balancer (ALB) y un Auto Scaling Group configurado para mantener 10 instancias EC2 corriendo con el servidor web NGINX. La solución está diseñada de tal forma que la página web sigue accesible en internet incluso si quedara una sola instancia activa
